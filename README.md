## How to Use

1. **Invite ZION** to your server.  
2. Join a voice channel.  
3. Use simple commands to control the music:  

| Command       | Description                                 |
| --------------|---------------------------------------------|
| `.play <url or title>` | Play a song or add it to the queue |
| `.volume`              | Pause the current song             |
| `.stop`                | Stop the music                     |
